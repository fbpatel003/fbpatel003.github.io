2. Description of Service

TGDrive is a free, open-source, browser-based web application that provides a user interface for managing files stored on Telegram's servers using the official Telegram MTProto API. TGDrive itself does not host, store, or process any user files. All file storage is provided entirely by Telegram's infrastructure. TGDrive functions as a frontend interface only.
