About TGDrive

TGDrive is a free, open-source, browser-based cloud storage application that uses your existing Telegram account as unlimited file storage — with no subscriptions, no server costs, and no middleman between you and your data.

What is TGDrive?

Telegram is one of the world's largest messaging platforms with globally distributed servers. Every Telegram account comes with effectively unlimited cloud storage — files sent to your "Saved Messages" or private channels persist on Telegram's servers indefinitely, accessible from any device worldwide.

TGDrive provides a familiar Google Drive-style interface on top of this storage. You create folders, upload files of any type up to 2 GB each, preview images in the browser, and stream videos — all through a clean, fast web interface with no installation required.

Unlike traditional cloud storage services such as Google Drive, Dropbox, or OneDrive, TGDrive has no backend server of its own. Your browser connects directly to Telegram's servers using the official Telegram MTProto API — the same protocol used by Telegram's own mobile and desktop apps. Your files, credentials, and session data never pass through TGDrive infrastructure because there is no TGDrive infrastructure.

How It Works

🔌 Direct MTProto Connection

Your browser opens a WebSocket connection directly to Telegram's servers using the MTProto protocol. No proxy, no relay, no middleman. Data flows between your browser and Telegram only.

📁 Folders as Private Channels

Each folder you create in TGDrive becomes a private Telegram channel tagged [TGDrive]. Files are stored as document attachments inside these channels with a JSON metadata caption preserving the original filename, size, and MIME type.

🔐 Local Session Storage

Your Telegram API credentials and session token are stored exclusively in your browser's IndexedDB — a local browser database. They are never transmitted to any TGDrive server because there isn't one.

🎬 True Video Streaming

Videos are streamed using a Service Worker that intercepts HTTP Range requests from the browser's native video player. Only the chunks needed for the current playback position are fetched from Telegram — memory-efficient, like professional streaming services.

🖼️ Image Thumbnails

Image previews are generated using Telegram's built-in thumbnail system. Thumbnails are cached in IndexedDB and automatically evicted after 30 days to keep local storage clean.

☁️ Telegram as the Backend

Telegram's global infrastructure handles all storage, redundancy, and availability. TGDrive is purely a frontend UI layer — a browser interface over the Telegram API with no server-side component.

Architecture

Your Browser
├── React UI (TGDrive interface)
├── GramJS (Telegram MTProto client — WebSocket)
├── Service Worker (video range request handler)
├── IndexedDB (session + metadata + thumbnail cache)
└── WebSocket ──────────────────► Telegram Servers
                                  └── Your files stored here permanently

# No TGDrive server exists between you and Telegram

Key Features

  • Unlimited storage — Telegram imposes no quota on user accounts
  • Up to 2 GB per individual file
  • Folder organisation with create, rename, and delete support
  • Multi-file drag-and-drop upload with real-time progress
  • Image preview with lazy-loaded thumbnails cached locally
  • Video streaming via Service Worker — plays while downloading, no full-file buffering
  • File search across all folders and files stored in local cache
  • Persistent login — session stored in IndexedDB, survives page refreshes
  • No auto-download on Telegram mobile — files stored as documents, not media
  • Responsive design — works on desktop and mobile browsers
  • Night theme — easy on the eyes for extended use

Frequently Asked Questions

Is TGDrive affiliated with Telegram?

No. TGDrive is an independent, community-built open-source project. It uses Telegram's official public API but is not affiliated with, endorsed by, or officially supported by Telegram Messenger Inc.

What is the storage limit?

Telegram does not impose a storage quota on user accounts. Individual files are limited to 2 GB each — significantly higher than the 100 MB limit on free Google Drive uploads via API. You can upload as many files as you want across as many folders as you need.

What happens if I clear my browser data?

Your local session, cached file metadata, and thumbnails are cleared. You will need to log in again with your Telegram credentials. However, all your files remain permanently on Telegram's servers and will reappear immediately after re-authentication — nothing is lost.

Is my data safe?

Your files are as safe as your Telegram account. TGDrive never sees your data — everything flows directly between your browser and Telegram's infrastructure. We strongly recommend enabling two-factor authentication on your Telegram account for additional security.

Can I use TGDrive for free?

Yes. TGDrive is completely free to use. The service is sustained by non-intrusive Google AdSense advertisements displayed on the drive interface. The login, about, privacy, and terms pages are ad-free.

Does TGDrive work on mobile browsers?

Yes. TGDrive is a fully responsive progressive web application and works on Android and iOS browsers. Chrome and Firefox provide the best experience. Safari on iOS has some limitations with Service Worker video streaming.

Open Source

TGDrive is fully open-source under the MIT License. You can inspect every line of code, self-host your own instance, or contribute improvements. Since TGDrive has no backend, self-hosting means simply serving the static build output — deployable for free on GitHub Pages, Vercel, Netlify, or any static file host.
