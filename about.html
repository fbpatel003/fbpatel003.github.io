About TGDrive

TGDrive is a free, open-source, browser-based cloud storage application that uses your existing Telegram account as unlimited file storage — with no subscriptions, no server costs, and no middleman between you and your data.

What is TGDrive?

Telegram is one of the world's largest messaging platforms with globally distributed servers. Every Telegram account comes with effectively unlimited cloud storage — files sent to your "Saved Messages" or private channels persist on Telegram's servers indefinitely, accessible from any device worldwide.

TGDrive provides a familiar Google Drive-style interface on top of this storage. You create folders, upload files of any type up to 2 GB each, preview images in the browser, and stream videos — all through a clean, fast web interface with no installation required.

Unlike traditional cloud storage services such as Google Drive, Dropbox, or OneDrive, TGDrive has no backend server of its own. Your browser connects directly to Telegram's servers using the official Telegram MTProto API — the same protocol used by Telegram's own mobile and desktop apps. Your files, credentials, and session data never pass through TGDrive infrastructure because there is no TGDrive infrastructure.

How It Works

🔌 Direct MTProto Connection

Your browser opens a WebSocket connection directly to Telegram's servers. No proxy, no relay, no middleman. Data flows between your browser and Telegram only.

📁 Folders as Private Channels

Each folder you create becomes a private Telegram channel tagged [TGDrive]. Files are stored as document attachments with a JSON metadata caption preserving filename, size, and MIME type.

🔐 Local Session Storage

Your Telegram API credentials and session token are stored exclusively in your browser's IndexedDB. They are never transmitted to any TGDrive server.

🎬 True Video Streaming

Videos stream via a Service Worker intercepting HTTP Range requests — fetching Telegram chunks on demand. Only what's needed for current playback position is loaded.

🖼️ Image Thumbnails

Previews use Telegram's built-in thumbnail system, cached in IndexedDB and auto-evicted after 30 days to keep local storage clean.

☁️ Telegram as the Backend

Telegram's global infrastructure handles all storage, redundancy, and availability. TGDrive is purely a browser-side UI layer over the Telegram API.

├── React UI (TGDrive interface)

├── Service Worker (video range request handler)
├── IndexedDB (session + metadata + thumbnail cache)
└── WebSocket ──────► Telegram Servers
                      └── Your files stored here permanently

# No TGDrive server exists between you and Telegram

Key Features

  • Unlimited storage — Telegram imposes no quota on user accounts
  • Up to 2 GB per individual file
  • Folder organisation with create and delete support
  • Multi-file drag-and-drop upload with real-time progress bars
  • Image preview with lazy-loaded thumbnails cached locally
  • Video streaming — plays while downloading, no full-file buffering
  • File search across all folders and files in local cache
  • Persistent login — session survives page refreshes
  • Files stored as documents — no auto-download on Telegram mobile
  • Responsive — works on desktop and mobile browsers

Frequently Asked Questions

Is TGDrive affiliated with Telegram?

No. TGDrive is an independent open-source project using Telegram's official public API. Not affiliated with or endorsed by Telegram Messenger Inc.

What is the storage limit?

Telegram imposes no storage quota. Individual files are limited to 2 GB each. You can upload as many files across as many folders as you need.

What if I clear browser data?

Local session and cache are cleared. You re-login. All files remain permanently on Telegram's servers and reappear immediately after re-authentication.

Is it free?

Yes. TGDrive is free, sustained by non-intrusive ads. No subscription, no hidden costs.

Open Source

TGDrive is open-source under the MIT License. Self-hosting means serving the static build — deployable free on GitHub Pages, Vercel, or Netlify.
